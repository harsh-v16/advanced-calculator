# Advanced Calculator (Python)

## Features:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)

## How to Use:
1. Enter the first number
2. Enter an operator (+, -, *, /, %)
3. Enter the second number
